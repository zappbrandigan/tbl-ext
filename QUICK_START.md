# Quick Start Guide

## Install the Extension in 3 Easy Steps

### Step 1: Open Chrome Extensions Page
- Open Google Chrome
- Type `chrome://extensions/` in the address bar and press Enter
- Or click the three dots menu → More Tools → Extensions

### Step 2: Enable Developer Mode
- Toggle the "Developer mode" switch in the top-right corner

### Step 3: Load the Extension
- Click the "Load unpacked" button
- Navigate to this project folder
- Select the **`public`** folder
- Click "Select Folder"

## You're Done!

The extension icon should now appear in your Chrome toolbar.

## Try It Out

1. Visit any webpage with HTML tables (try Wikipedia, data tables, etc.)
2. Click the extension icon
3. See all tables detected on the page
4. Click any table to download it as CSV
